# Rock Paper Scissors Game

A simple command-line Rock Paper Scissors game written in Python.

## Features

- Play against computer
- Score tracking
- Input validation
- Support for short commands (r, p, s)

## How to Run

```bash
python script.py
```

## How to Test

```bash
python -m unittest test_script.py
```

## Game Rules

- Rock beats Scissors
- Paper beats Rock  
- Scissors beats Paper

## Commands

- `rock` or `r` - Play rock
- `paper` or `p` - Play paper
- `scissors` or `s` - Play scissors
- `quit` - Exit the game
